D O M (Document Object Model)

1. Meaning of DOM :

> The Document Object Model(DOM) is an application programming interface(API) for manipulating HTML documents.
> The DOM provides functions that allow to add, remove and modify parts of the document effectively.
> The DOM represents an HTML document as a tree of nodes.
> Each tag in a document is a node and each node has parent child relationships in the DOM tree.
> The html node is a parent node of head and body child node.

2. Methods of DOM :

> document.getElementById()

> document.getElementByName()

> document.getElementsByTagName()

> document.getElementsByClassName()

> document.querySelector()

> document.querySelectorAll()

3. Traversing Elements :

> To get the parent node: parentNode
> To get child Elements of node: firstElementChild, lastElementChild, childNodes
> To get next/prev Siblings: previousElementSibling / nextElementSibling
